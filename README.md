# Twitch Network Analytics

Network analysis project for Twitch streamers and communities.

## Project Structure

- `data/` - Raw and processed data
- `src/` - Source code modules
- `notebooks/` - Jupyter notebooks for exploration
- `results/` - Output graphs and metrics
- `tests/` - Unit tests

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Refer to the notebooks in the `notebooks/` directory for exploratory analysis and examples.
